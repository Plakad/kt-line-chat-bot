# LINE Chat Bot

This is a Python-based chat bot built using the LINE Messaging API. The bot can handle text messages, reply automatically, and support both personal and group chats. Itâ€™s deployed on Render.

## Features

- Responds to text messages.
- Supports both personal chats and group chats.
- Handles events such as message, follow, and join events.
- Deployed on Render for easy scalability.

## Requirements

To run this project, you'll need:

- Python 3.8+
- The following Python libraries (listed in `requirements.txt`):
  - `Flask`
  - `line-bot-sdk`
  - `gunicorn`
